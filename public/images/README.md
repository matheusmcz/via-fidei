# Imagens

Esta pasta contém as imagens das igrejas.

## Placeholder

Quando uma igreja não possui imagem (`imageUrl` ausente), o sistema utiliza automaticamente o arquivo `placeholder.svg`.

## Estrutura sugerida

```
/public/images/
  placeholder.svg          # Imagem padrão
  churches/                # Fotos das igrejas (futuro)
    sao-jose-operario.jpg
    nossa-senhora-...jpg
```
